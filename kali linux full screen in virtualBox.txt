# Kali Linux Full Screen in VirtualBox
# From: https://www.youtube.com/watch?v=6LaHG8SC8n4

Open VirtualBox
    +-- Boot to your kali linux
    +-- On Status Bar
    |    +-- Go to Devices 
    |    |    +-- Insert Guest Adtional CD image.
    |    |         # A removeable volume (ISO CD Image) should appear in your desktop.
    +-- Open and Copy all contents of ISO file to any location (Desktop).
    +-- Open terminal on the folder you copied these file.
    |    # Enter following commands as root.
    |
    |       chmod +x VBoxLinuxAdditions.run
    |       ./VBoxLinuxAdditions.run
    |
    +-- Remove all the copied files and Unmount/Eject the iso file.
    |
    +-- Reboot
